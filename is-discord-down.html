Why is Discord not working?

1. A Discord-side outage

Discord has high availability but does go down occasionally. When it happens, the entire platform fails at once and discordstatus.com updates within minutes.

2. Voice server region issue

Discord voice runs on region-specific servers (US East, US West, Europe, Asia, etc.). Any one region can fail independently while the others work fine. Server admins can switch the region in Server Settings → Overview.

3. Network block

Schools, offices, and public Wi-Fi frequently block Discord by default. If Discord works on mobile data but not Wi-Fi, that's why.

4. Desktop app glitch

The Discord desktop client occasionally gets stuck on "connecting" or "reconnecting" when the web version works fine. Fully quit and restart the app — not just close the window.

5. API rate limit

If you're using a Discord bot or integration, hitting the API rate limit causes "something went wrong" errors. Wait a few minutes and the limit resets.

6. Account action

A muted server, disabled account, or banned user sees Discord errors that look like a full outage. Check if you can log into a different account.

What to do if Discord is only down for you

  • Try Discord in a browser. discord.com in Chrome or Firefox. Bypasses desktop-app bugs.
  • Fully quit the desktop app. Right-click the tray icon → Quit Discord. Then relaunch.
  • Change voice region. If voice fails, server admin can change region in Server Settings → Overview.
  • Switch networks. If Discord works on mobile data but not Wi-Fi, your network is blocking it.
  • Use a VPN. For school / office blocks, a VPN typically restores access.
  • Clear Discord cache. %appdata%/discord/Cache on Windows.
