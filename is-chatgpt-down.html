Why is ChatGPT not working?

1. An OpenAI outage

ChatGPT has frequent partial outages, especially during peak hours. status.openai.com lists active incidents. When OpenAI has an issue, ChatGPT, GPT-4, DALL-E and the API are usually affected together.

2. Rate limit (free tier)

Free ChatGPT has daily message caps. If you see "too many requests" or "please try again later", you've hit the limit — it's not an outage. Resets every few hours.

3. GPT-4 message limit (Plus tier)

ChatGPT Plus has a separate, stricter limit on GPT-4 messages (updated by OpenAI periodically). Hitting it blocks GPT-4 but GPT-3.5 still works.

4. Capacity throttling

During high demand, OpenAI throttles free users first. If chat.openai.com shows "ChatGPT is at capacity", it's not down — it's overloaded. ChatGPT Plus bypasses this.

5. Country restriction

OpenAI blocks several countries. If ChatGPT fails only in certain regions, a VPN to an allowed country (US, UK, EU) typically restores access.

6. Account or browser issue

A stale session or corrupt cookies can mimic an outage. Log out and back in, or try an incognito window.

What to do if ChatGPT is only down for you

  • Check for rate-limit messages. If you see "too many requests", wait — not an outage.
  • Try an incognito window. Bypasses cached cookies and extensions.
  • Log out and back in. Refreshes your session.
  • Clear cookies for chat.openai.com and openai.com.
  • Switch to GPT-3.5. If GPT-4 fails, free GPT-3.5 may still work.
  • Try the OpenAI API or a different client. The API sometimes works when ChatGPT.com itself is slow.
